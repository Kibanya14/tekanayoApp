# 🏪 Système de Niches et Icônes - TekanayoApp

## 🎯 Vue d'Ensemble

Le système de niches permet à chaque vendeur de choisir sa spécialité lors de l'enregistrement. Chaque niche dispose de ses propres icônes FontAwesome pour catégoriser les produits.

## 💾 Base de Données

### Table `seller_shops`

```sql
CREATE TABLE seller_shops (
    id INTEGER PRIMARY KEY,
    name VARCHAR(140) NOT NULL,
    slug VARCHAR(160) UNIQUE NOT NULL,
    category_niche VARCHAR(60) DEFAULT 'shopdivers',  -- Niche choisie
    ...
);
```

## 🔧 Utilisation dans le Code

### Backend (Python/Flask)

```python
# Récupérer les icônes d'une niche
niche = "pharmacie_soin"
icons = CATEGORY_NICHES.get(niche, [])
# Retourne: ['fa-briefcase-medical', 'fa-pills', ...]

# Vérifier si une niche est valide
from backend.apps import NICHE_CHOICES

valid_niches = dict(NICHE_CHOICES)
if niche in valid_niches:
    print(f"Niche valide: {valid_niches[niche]}")
```

### Frontend (JavaScript)

```javascript
// Récupérer les icônes depuis les données injectées
const nicheIcons = window.NICHE_ICONS || {};
const pharmacyIcons = nicheIcons['pharmacie_soin'];
// Retourne: ['fa-briefcase-medical', 'fa-pills', ...]

// Afficher les icônes d'une niche
pharmacyIcons.forEach(icon => {
    const iconHtml = `<i class="fas ${icon}"></i>`;
    container.appendChild(iconHtml);
});
```

### Template Jinja2

```html
<!-- Afficher toutes les icônes d'une niche -->
{% set niche_icons = CATEGORY_NICHES.get(shop.category_niche, []) %}
<div class="niche-icons">
    {% for icon in niche_icons %}
    <i class="fas {{ icon }}" title="{{ icon }}"></i>
    {% endfor %}
</div>
```

## 🚀 Comment Ajouter une Nouvelle Niche

### 1. Dans `backend/apps.py`

```python
CATEGORY_NICHES = {
    # ... autres niches
    "nouvelle_niche": [
        "fa-icon-1", "fa-icon-2", "fa-icon-3",
        "fa-icon-4", "fa-icon-5", "fa-icon-6",
        "fa-icon-7", "fa-icon-8", "fa-icon-9",
        "fa-icon-10", "fa-icon-11", "fa-icon-12",
        "fa-icon-13", "fa-icon-14", "fa-icon-15"
    ],
}

NICHE_CHOICES = [
    # ... autres niches
    ("nouvelle_niche", "Nom de la nouvelle niche"),
]
```

### 2. Mettre à jour le modèle (si nécessaire)

```python
# backend/models.py
class SellerShop(db.Model):
    category_niche = db.Column(db.String(60), default="shopdivers")
    # Aucun changement nécessaire si le champ existe déjà
```

### 3. Appliquer les migrations

```bash
flask db migrate -m "Ajout nouvelle niche"
flask db upgrade
```
