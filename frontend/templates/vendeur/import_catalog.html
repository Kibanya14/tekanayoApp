Format requis

Votre fichier doit contenir une ligne d'en-tête avec les colonnes suivantes :

Colonne Accepte Exemple Obligatoire?
name product_name, titre Téléphone Samsung S24 OUI
price prix, product_price 99.99 OUI
category categorie, category_id Électronique Non (défaut: Autres)
description descriptif Smartphone 5G dernière génération Non
quantity stock, quantite 50 Non (défaut: 0)
compare_price prix_compare 129.99 Non
is_active actif, enabled true / false Non (défaut: true)
is_featured en_vedette, featured true / false Non (défaut: false)

Note: Les noms de colonnes ne sont pas sensibles à la casse. "Nom du produit", "name", "Product Name" sont tous acceptés.
