Aperçu

À faire

  • Vérifier les doublons avant import
  • Utiliser prix décimaux: 99.99 (pas 99,99)
  • Catégories seront créées auto si manquantes
  • Supprimer les lignes vides
  • Sauvegarder l'original avant modification

À éviter

  • Caractères spéciaux dans les noms
  • Prix négatifs ou illégaux
  • Espaces inutiles en début/fin
  • Fichiers trop volumineux (>100MB)
  • Importer sans vérifier les données
